Logical vs Comparison Operators

What Are Comparison Operators?

Definition: Comparison operators compare two values and return a Boolean result: either true or false.

Common Comparison Operators

OperatorDescriptionExampleResult
==Equal to (value)5 == "5"true
===Equal to (value & type)5 === "5"false
!=Not equal to5 != 4true
!==Not equal (value & type)5 !== "5"true
>Greater than10 > 7true
<Less than3 < 5true
>=Greater than or equal to5 >= 5true
<=Less than or equal to4 <= 3false

Example:

let age = 18;
console.log(age >= 18); // true
console.log(age == "18"); // true (in JS)

What Are Logical Operators?

Definition: Logical operators combine or invert Boolean values (usually from comparisons).

Common Logical Operators

OperatorDescriptionExampleResult
&&AND — both must be truetrue && truetrue
||OR — one must be truetrue || falsetrue
!NOT — inverts the result!truefalse

Example:

let age = 20;
let hasID = true;

if (age >= 18 && hasID) {
  console.log("You can enter.");
}

Key Differences

FeatureComparison OperatorsLogical Operators

Combined Example

let score = 85;

if (passed && hasCertificate) {
  console.log("You passed and are certified!");
